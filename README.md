# SHWCare Static Site (v3)

Static HTML/CSS site designed to mimic a clean Shopify-style wellness site while staying fully static.
- No PHI is collected on public pages.
- Use Start Intake to route consumers/providers to the next step.
- Replace logo/copy as needed.

## Deploy
Upload all files to the repo root and enable GitHub Pages (main / root).

## Branding
Edit assets/styles.css:
- --navy and --red are the brand accents.
- Replace /assets/shw-logo.svg with your official SHW logo when ready.
